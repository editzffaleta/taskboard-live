# Leituras obrigatorias — e2e-playwright

Antes de escrever testes e2e, leia:

1. `references/playwright-conventions.md` — padroes (seletores, espera, isolamento, auth).
2. `references/few-shots/playwright.config.example.ts` — baseURL, webServer, projects/setup.
3. `references/few-shots/auth.setup.example.ts` — login uma vez -> storageState.
4. `references/few-shots/register-login.e2e.example.ts` — fluxo critico de referencia.
5. Os `data-testid` reais das telas (alinhar com `frontend-next-config` / `frontend-specialist`).
6. O `spec-flow` (gate) para ligar `test:e2e` ao portao de qualidade.
