# Mockup de exemplo — D7 (grupos de permissão)

**Este é um exemplo DEMONSTRATIVO da convenção `mockups/<tela>/`** — substitua pelos mockups
reais do Claude Design antes de executar a change. Se a tela não tiver mockup, **delete a
subpasta** (mockup é condicional por tela).

- Um arquivo por tela (`mockup.html`, `.png`, o que o Claude Design gerar), nesta subpasta.
- Na execução: **reproduza fielmente o layout** do mockup e **substitua todo dado
  fake/placeholder por dado real do backend** — os dados marcados com `data-fake` no exemplo
  existem só para dar forma ao layout.
